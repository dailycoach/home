# DISC16 내부 QA 콘솔

접속 경로:

`/tests/disc16/qa.html`

이 페이지는 검색엔진에 노출하지 않는 내부 검수용 화면이다.

## 자동 검증 항목

- 16개 상황문항 존재 여부
- 각 문항의 D·I·S·C 보기 구성
- 보기 위치 순열의 유효성
- 기본유형 D·I·S·C 해설 필드 누락
- 12개 조합형 해설 존재 여부
- 조합형 해설 필드 누락
- 상대유형별 대화지침
- 4×4 교류주의 매트릭스
- 단일형 4개와 복합형 12개의 합성 결과코드
- 모든 합성응답의 총점 160점
- 0~10척도 범위

## 결과 미리보기

유형 버튼을 누르면 해당 유형이 나오도록 합성 응답 16개를 생성한다.
오른쪽 미리보기 화면에서 결과지를 바로 열어 다음을 검수한다.

- 결과 코드와 유형명
- 1·2·3순위
- 1순위 심층해설
- 1·2순위 조합 해설
- 좌표와 분포
- D·I·S·C 상세카드
- 대인관계 교류지침
- 코칭질문
- 모바일 스크롤과 카드 배치

## 주의

QA 유형 버튼은 브라우저의 `disc16:answers:v2` 값을 합성 응답으로 덮어쓴다.
실제 검사 응답을 보존해야 하는 브라우저에서는 사용하지 않는다.
검수 후 `QA 응답 삭제` 버튼으로 저장값을 제거한다.
