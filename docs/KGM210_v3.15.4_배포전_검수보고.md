## 2. 이번 회귀검수에서 발견한 오류

### 2.1 Playwright 웹서버 기준경로 오류

테스트 설정파일의 위치를 기준으로 웹서버가 실행되어 `/tests/kgm210/index.html`을 찾지 못했다.

조치:

- 웹서버 작업경로를 저장소 루트로 고정했다.

### 2.2 성장여정 닫기 후 재방문 카드 미복원

새로고침 후 7주 화면을 열었다가 닫으면 첫 화면은 보이지만 `이번 주 대화 준비` 카드가 다시 생성되지 않는 경우가 있었다.

조치:

- 돌아가기 직후 첫 화면 또는 성장여정 화면의 진입카드를 재구성한다.
- 현재 주차, KINGDOM 단계, 지난 문장과 작은 행동을 다시 표시한다.

### 2.3 동적으로 추가된 7일 표현 미교체

초기화 이후 삽입된 텍스트는 기존 전체화면 치환만으로 처리되지 않았다.

조치:

- MutationObserver가 새로 추가된 텍스트 노드를 즉시 처리하도록 변경했다.
- `기존 7일 기록` 안내영역은 역사자료이므로 치환 대상에서 제외한다.

### 2.4 성장여정 열기 이벤트 중복

UI의 기본 클릭 처리와 QA 보강 클릭 처리가 함께 실행되면, 진입 출처가 덮어써져 돌아가기 위치가 달라질 수 있었다.

조치:

- 캡처 단계에서 한 번만 진입을 처리한다.
- `intro`, `result`, `journey` 출처를 명시적으로 보존한다.

### 2.5 이전 주차 열람 시 현재 주차가 후퇴

과거 주차를 다시 열어 수정할 때 `currentWeek` 자체가 과거 주차로 변경될 가능성이 있었다.

조치:

- 현재 진행주차와 화면에서 보고 있는 주차를 분리했다.
- 과거 기록을 수정해도 해금된 현재 주차는 유지된다.
- 자동저장은 화면에서 보고 있는 주차에 적용된다.

### 2.6 QA 스크립트 strict mode 문법 오류

`'use strict'`가 있는 함수에서 기본 매개변수를 사용해 Node 문법검사를 통과하지 못했다.

조치:

- 기본 매개변수를 제거하고 함수 내부에서 숫자값을 정규화했다.
