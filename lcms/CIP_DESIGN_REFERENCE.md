## Locked visual system

- Background: `#eef4fb` with soft radial light gradients
- Primary navy: `#071c43`
- Secondary navy: `#0d2d67`
- Text: `#0a1734`
- Muted text: `#6d7a96`
- Green accent: `#16d66f`
- Mint accent: `#7df2c2`
- Orange accent: `#ff8b18`
- Orange secondary: `#ffb156`
- Blue accent: `#257cff`
- Purple accent: `#443b9d`
- Card: white
- Main radius: `28px`
- Card radius: `18–24px`
- Main shadow: `0 20px 55px rgba(13, 29, 71, .15)`
- Soft shadow: `0 12px 30px rgba(15, 31, 71, .09)`

## Layout language

- Light mineral-blue canvas
- Deep navy title blocks and caps
- White rounded cards with thin blue-gray borders
- Orange → green → blue → purple progression for lifecycle and capability flows
- Dark navy used only for opening/closing emphasis panels, not as the default whole-page background
- Pretendard / Apple SD Gothic Neo / Noto Sans KR
- Large bold Korean headings, compact English eyebrow labels
- Visuals emphasize connected paths, lifecycle curves, feedback loops, verified assets, portfolios, and dashboards
